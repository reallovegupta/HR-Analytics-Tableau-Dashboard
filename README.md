# HR Analytics Dashboard (Tableau)

**Author:** Abhishek Gupta

## Project Overview

This project presents an **HR Analytics Dashboard built using Tableau** to analyze employee data and identify workforce trends such as attrition, job satisfaction, and demographic patterns.

The dashboard helps HR teams understand employee behavior and improve decision-making using interactive visualizations.

## Dashboard Insights

### Employee Attrition

The dashboard highlights overall attrition and attrition rate across departments and demographics.

### Department Analysis

Attrition is analyzed across departments such as:

* HR
* Research & Development
* Sales

### Age Group Distribution

Employee counts are segmented across age groups including:

* Under 25
* 25–34
* 35–44
* 45–54
* Over 55

### Job Satisfaction Analysis

Job satisfaction ratings are analyzed across multiple job roles to identify areas needing improvement.

### Education Field Analysis

Attrition trends are analyzed across different education fields including:

* Life Sciences
* Medical
* Marketing
* Technical Degree
* Human Resources

### Gender-Based Attrition

Attrition rate is also analyzed across gender and different age groups to understand workforce patterns.
